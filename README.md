# Learning Playwright

This repository contains code examples and exercises for learning Playwright automation testing.

## Project Structure

- **Chapter_01_Basics**: Contains basic JavaScript and Node.js setup verification scripts.

## Prerequisites

- Node.js installed (v14 or higher recommended)

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/AuTeLipi/LearningPlaywright.git
   ```
2. Navigate to the project directory:
   ```bash
   cd LearningPlaywright
   ```

## Running Scripts

To run any script, use node:

```bash
node Chapter_01_Basics/01_Basics_PrintOutput.js
```
